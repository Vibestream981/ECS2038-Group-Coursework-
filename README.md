# ECS2038-Group-Coursework-

The files included all need to be in the same folder when ran, as some reference other ones, and they should be ran in the following order:

1) parameters.py (Parameters for all values)
2) nonlinear_dynamics.py (Use these values to describe the formulaes and physics)
4) linearisation.py (Values to build the linear model)
5) analysis_and_plots.py 
6) controller.py (Designs PID)
7) simulation_engine.py (Run simulations by combining PID and nonlinear)
8) test_scenarios.py (Saving the plots of different scenerio)

It may not run correctly if they are not all ran in that order whilst being in the same folder.
